# CVSS Overview


- **Range**: 0.0 to 10.0
- **Severity mapping**:
  - Low: 0.1–3.9
  - Medium: 4.0–6.9
  - High: 7.0–8.9
  - Critical: 9.0–10.0

Factors include:
- Attack Vector (AV)
- Attack Complexity (AC)
- Privileges Required (PR)
- User Interaction (UI)
- Scope (S)
- Confidentiality / Integrity / Availability impact (C/I/A)

Use CVSS scores reported by scanners to prioritize fixes.
